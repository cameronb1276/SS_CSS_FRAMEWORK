SS CSS Framework Documentation

A class-based, Bootstrap-like CSS framework for business websites, SloanSites builder surfaces, and hardened published client output.

TokensColors, type, spacing, radius, shadows, motion, and layers. UtilitiesContainers, grids, spacing, sizing, display, text, and safety helpers. ComponentsButtons, cards, forms, alerts, navs, tables, accordions, and more. SectionsBusiness website section primitives and local-business patterns. Builder CanvasCanvas, device previews, editable states, handles, and drop zones. Builder UIEditor shell, sidebars, inspector, layers, assets, and statuses. ThemesClient branding wrappers, variables, and preset theme helpers. Published SafetyHardening for long text, media, embeds, tables, forms, and print.
